# Story 13.5: Coordinated Releases

Status: ready-for-dev

## Story

As a **developer**, I want **to release multiple repositories together**, so that **dependent packages are released atomically**.

## Acceptance Criteria

1. **AC1**: Define release groups
2. **AC2**: Version bump across repos
3. **AC3**: Coordinated changelog
4. **AC4**: Atomic release (all or none)
5. **AC5**: `orchestrate release --repos <list> --version <version>`
