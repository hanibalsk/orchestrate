# Story 13.4: Cross-Repository PRs

Status: ready-for-dev

## Story

As a **developer**, I want **to manage related PRs across repositories**, so that **coordinated changes can be merged together**.

## Acceptance Criteria

1. **AC1**: Link PRs across repositories
2. **AC2**: Track linked PR status
3. **AC3**: Coordinate PR merging order
4. **AC4**: `orchestrate pr link --pr <number> --repo <repo>`
5. **AC5**: Block merge if dependencies not merged

## Tasks / Subtasks

- [ ] Task 1: PR linking
- [ ] Task 2: Status tracking
- [ ] Task 3: Merge ordering
- [ ] Task 4: Dependency blocking
- [ ] Task 5: Write tests
